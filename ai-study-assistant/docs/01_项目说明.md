## 一、项目简介

| 项 | 内容 |
|---|---|
| **项目名** | AI 学习助手（可自取，如 `StudyPal`、`思伴`） |
| **一句话** | 一个能陪你刷题、讲解知识点、查笔记、记录错题的私人 AI 助教 |
| **目标用户** | 大学生 / 考研党 / 自学者 |
| **核心价值** | 和 ChatGPT 不同，它**认识你**——记得你学过的、错过的、想问的 |

**对比通用 AI 的差异化**：

| 维度 | ChatGPT | AI 学习助手 |
|---|---|---|
| 是否记得你的错题 | ❌ | ✅ |
| 是否能查你笔记 | ❌ | ✅ |
| 能否出"举一反三"题 | ❌ | ✅ |
| 能否看学习数据 | ❌ | ✅ |
| 个性化推荐 | ❌ | ✅ |

## 二、技术栈（选最稳的，别折腾）

### ✅ v1.0 必选（必须掌握）

| 模块 | 技术 | 为什么选它 |
|---|---|---|
| 编程语言 | Python 3.10+ | 你正在学 |
| Web 框架 | **FastAPI** | 比 Flask 快，自动生成 API 文档 |
| LLM 调用 | 智谱 GLM-4 / 通义千问 / DeepSeek | 国产便宜，新用户有免费额度 |
| 聊天界面 | **Gradio** | 200 行代码出界面，专为 AI 设计 |
| 向量数据库 | **ChromaDB** | 轻量、本地跑、零配置 |
| 嵌入模型 | bge-small-zh（智源） | 中文效果好，免费，本地跑 |
| 笔记解析 | PyPDF2 + python-docx | 处理 PDF 和 Word 笔记 |
| 关系数据库 | SQLite | 存错题、学习记录，零配置 |
| 部署 | Hugging Face Spaces | 免费额度够用，海外可访问 |

## 三、功能清单

### 🎯 v1.0 必做功能（4 个）

#### F1. 多轮对话 ✅
- 记住对话上下文
- 支持"新建对话"切换话题
- 显示历史对话列表
- 支持清空上下文

#### F2. 学习模式（核心卖点）⭐
- **错题本**：录入题目 + 你的答案，AI 评分 + 讲错因
- **AI 出题**：基于错题自动生成 3 道类似题
- **知识点讲解**：用"老师讲题"的口吻回答，不只是给答案
- **练习模式**：按知识点 / 错误率筛选错题

#### F3. 笔记问答（RAG 基础）⭐
- 上传 PDF / Word 笔记
- AI 读取后基于笔记内容回答
- 显示答案引用了哪段原文
- 例："我数据结构笔记里讲红黑树的部分怎么说的？"

#### F4. 学习数据看板
- 累计学习时长
- 错题数量趋势
- 高频错题知识点 Top 5
- 学习日历（GitHub 绿点风格）

### 🌟 v1.1 选做功能

- 多用户登录
- 错题导出 Anki
- 移动端适配
- 语音输入

## 六、关键技术点（提前看，不慌）

### 1️⃣ LLM 调用最小示例（智谱 GLM-4）

```python
from openai import OpenAI

client = OpenAI(
    api_key="你的API_KEY",
    base_url="https://open.bigmodel.cn/api/paas/v4/"
)

response = client.chat.completions.create(
    model="glm-4-flash",  # 免费模型
    messages=[{"role": "user", "content": "你好"}]
)
print(response.choices[0].message.content)
```

### 2️⃣ Gradio 聊天最简版

```python
import gradio as gr

def chat(message, history):
    return "AI 回复：" + message

demo = gr.ChatInterface(fn=chat, title="我的 AI 学习助手")
demo.launch()
```

### 3️⃣ ChromaDB 检索最简版

```python
import chromadb

client = chromadb.PersistentClient(path="./data/vector_db")
collection = client.get_or_create_collection("notes")

# 添加文档
collection.add(
    documents=["红黑树是自平衡的二叉搜索树...", "快速排序平均复杂度 O(nlogn)"],
    ids=["1", "2"]
)

# 检索
results = collection.query(
    query_texts=["什么是红黑树"],
    n_results=2
)
print(results)
```

### 4️⃣ 错题评分 Prompt 模板

```
你是一位耐心的老师。学生做了一道题，请：
1. 判断对错
2. 如果错，指出错误原因
3. 给出正确答案
4. 总结这道题考察的知识点

题目：{question}
学生的答案：{student_answer}
正确答案：{correct_answer}

请用 JSON 格式返回：
{
  "is_correct": true/false,
  "feedback": "讲评...",
  "knowledge_point": "知识点",
  "similar_topics": ["相关知识点1", "相关知识点2"]
}
```

## 八、可能踩的坑（提前避雷 🚧）

| 坑 | 解决方案 |
|---|---|
| 💸 API 余额耗尽 | 用 `glm-4-flash` 免费模型，别用 pro |
